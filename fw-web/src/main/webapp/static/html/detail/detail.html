所谓习惯就是“不依赖意志或毅力，把自己想要持续的事情引导到如每天刷牙般轻松的状态”。其实对于大脑而言，没有所谓的好习惯和坏习惯的分别，习惯就是在脑中设定的在一定时期内重复进行某项行动的程序而已。 举 个栗子来说，早睡早起对我而言并不是难事，因为我本身不喜欢也不习惯熬夜，过了11点脑子自动进入待机状态，但到了早晨6点左右会自动开机。但对有些习惯 晚睡晚起的伙伴而言，刻意早睡早起不但让人感到痛苦，而且白天容易疲惫，效果不佳。如果“夜猫子”们可以利用好晚上安静而高效的时间，那晚睡晚起对于他们 而言就是好习惯。 2为什么我们容易产生“三分钟热度”？ 我曾经尝试学习过很多东西，如油画，拉丁舞，手风琴，钢琴，唱歌，法语，口译.....但最终都没能走到终点站。为什么？ 对于人类而言，我们习惯保持在固定的状态，因为这样能让我们感到舒适和安全，变化则被视为是一种威胁。正因为身体对“培养新习惯”的变化感受到了威胁，于是开始不断反抗，也就是“三分钟热度”现象的内部机制。认识到这一点后，我们就要有针对性的采取措施，让大脑认为这项新习惯“跟往常一样”就好了。 3习惯的三种分类 关于习惯养成有很多理论，21天养成一个好习惯，三个月养成习惯，10000小时定律等，但事实证明对于不同的习惯，所需的时间是不同的。 对于行为习惯，培养时间大约为一个月，主要是一些每天规律的行为如写日记、整理、读书等。对于身体习惯如减肥、运动、早起、戒烟等，由于这种变化对人的影响比较大，通常需要三个月。对于思考习惯如逻辑性思考能力、批判性思维、创意能力等，通常和我们的性格有关，身体对于变化所产生的抵抗也最强烈，需要六个月来养成。 4培养习惯的三个阶段 习惯培养主要分为三个阶段，阶段一：反抗期（第1天~第7天）马上就想放弃，42%的人失败。阶段二：不稳定期（第8天~第21天）被预定事项或他人影响，40%的人失败。阶段三：倦怠期（第22天~第30天）逐渐感到厌烦，18%的人失败。 如何才能针对不同阶段， 有效采取措施从而养成良好的习惯呢？ 我们要学会以农业眼光培养习惯，即拥有长远眼光，培养不同类型的好习惯。在决定培养什么习惯之前，我们要弄清楚自己三年、五年后要成为什么样的人，达到怎么样的目标、工作和生活上取得怎么样的成绩，从而进行习惯的选择。通常而言，习惯分为短期、中期和长期习惯。短期习惯效果立竿见影，如整理、不吃零食、减少看手机的次数。中期习惯包括时间管理、写日记、考职业证等。长期习惯需要长时间的积累才能达到水滴石穿的效果，如扩展人脉，终身学习，健康管理。 5顺利培养习惯的三个阶段 反抗期：咬牙坚持下去 在阶段一，我们会有各种各样不舒适以及想放弃的感觉，如学了觉得没用，时间不够，内容太无趣，想要试试别的等。在这个阶段，我们完全可以忽略行动量或结果，而是将重心放在开始行动并且坚持每天行动。 假如我想养成晨跑的习惯，原先的目标是每天30分钟，但是当第3-4天的时候，心里会产生抗拒感，这时候我们要做的不是逼自己跑完30分钟，而是告诉自己，我可以选择跑5-10分钟或者走20分钟，但是不能不行动，过两天之后，我们就又能恢复正常的30分钟晨跑了。 在这个阶段，我们可以锁定一项习惯（切忌三心二意），坚持有效的行动（事情越简单越容易坚持下去）并且不过分在意结果。可采用的方法如“以婴儿学步开始”，从小的地方开始做，不追求完美，而是注重开始行动的过程。再如每天进行简单记录，让过程和结果可视化，增强自信心。其中最关键的一点还是坚持每天持续进行。 不稳定期：提高门槛，形成稳定机制 在阶段二，我们经常会被加班或其他突发事情所打扰，从而产生厌倦、怀疑等不良情绪。在这个阶段，我们要学会调高门槛，使得行为模式化，设立好例外规则，设定持续开关。 行为模式化指的是将自己需要做的事情细分，具体到何时何地何人，如每天晚上10-11点，关闭一切通讯工具，专心看书，一周2本书，2篇书评。 设定例外规则是为了让计划保持弹性，即凡事都有Plan B。例如本来周三晚上准备去健身房上瑜伽课的，但是由于临时有事需要加班去不了，那我们的Plan B可以是在家附近夜跑30分钟或者室内跟着Keep软件健身。凡事提前想好PlanB，可以让我们更轻松地应对变化带来的不愉悦感。 文／怦然心动的人生（简书作者） 原文链接：http://www.jianshu.com/p/15d04affd4d8
